NMF Immune signatures

First, let’s look at the correlation between our immune signatures and their relationship with CCS. We’ll find the strongest signal is from variance in tumor content in the sample.

Stroma-Tumor-Immune classification

Next, we compare our immune signatures with the Stroma-Tumor-Immune (STIM) classification of Martin-Serrano et al. 2023. This classifier was trained to predict iCCAs so we subset to this population only. When we noticed the strong segregation specific STIM classes with CCS, we also enquired into whether tumor content might impact inference.

labels_df.raw <- fread(file.path(basedir,"/results/classified.FF.txt"))
stim.class <- labels_df.raw  %>% dplyr::select( Sample | starts_with("martinserrano") ) 
stim.class[stim.class == 'Hepati Stem-like'] <- 'Hepatic Stem-like'
  
stim.prob <- left_join(H_t_labs, stim.class, by=c('sample'='Sample')) %>% filter( !is.na(rna_class))

timeline.outcome <- fread(file.path(secdir,"/results/timeline.outcome.txt"))
timeline.outcome$BTCID <- gsub('_','',timeline.outcome$BTC_ID)
iccas <- timeline.outcome$BTCID[timeline.outcome$Location == 'Intrahepatic_CCA_iCCA']

stim.prob$content.bin <-NA
stim.prob$content.bin[stim.prob$cellularity <= 1 ] <- 'High (>=85%)'
stim.prob$content.bin[stim.prob$cellularity < 0.85 ] <- 'Mid (65-85%)'
stim.prob$content.bin[stim.prob$cellularity <= 0.65 ] <- 'Low (<=65%)'


transition_df.1 <- stim.prob %>% filter(!is.na(martinserrano.class) &  !is.na(rna_class) & BTCID %in% iccas) %>% group_by(martinserrano.class, rna_class) %>% tally()

colnames(transition_df.1) <- c("STIM", "CMS", "Freq")

transition.max <- transition_df.1 %>% group_by(STIM) %>% filter(Freq == max(Freq) ) 

transition_df.1$STIM.f <- factor(transition_df.1$STIM, 
                                  levels=transition.max$STIM[order(transition.max$CMS)])


ggplot(transition_df.1,
